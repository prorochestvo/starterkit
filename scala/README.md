# Plans

Markdown plan files for non-trivial work. See `CLAUDE.md` → **Planning Workflow**
for naming, lifecycle, and the agent pipeline that produces and consumes these
files.

```
plans/
├── NNN-task-slug.md     # active / in-progress
├── completed/           # YYMMDD.NNNN.slug.md
└── history/             # archived or cancelled
```
